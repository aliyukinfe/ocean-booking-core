# Ocean Booking Core

Custom WordPress plugin for events, guides, booking integration settings, event metadata, contact form, and event schema.

## WP Pusher

Install this repository as a **plugin** in WP Pusher. The plugin files live at the repository root and do not require a build step.

Required companion theme: `ocean-booking-theme`.

## Setup

1. Activate the plugin.
2. Go to **Settings > Ocean Booking**.
3. Configure the booking provider and integration mode.
4. Add events under **Events** and guides under **Guides**.
5. Publish or translate the starter pages created on activation.
